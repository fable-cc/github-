# /fable-brainstorm

## 用法

```
/fable-brainstorm <话题或关键词>
```

## 示例

```
/fable-brainstorm 35岁职场危机
/fable-brainstorm 知识付费还能做吗
/fable-brainstorm 为什么越省钱越穷
```

## 执行流程

1. 用冶炼矩阵（人性底层×权力博弈×变现路径×反常识）扫描话题
2. 每个维度找到国学锚点 + 心理锚点
3. 输出3-5个独特角度，附金句雏形和变现方向
4. 对最佳角度给出第一段示范

## 相关

- `fable-style` 技能 — 确定角度后的写作执行
