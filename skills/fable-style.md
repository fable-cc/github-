# 景一风格改写引擎

## When to Use

- 用户说"改写成我的风格""去AI味""冶炼一下""景一风格"
- 任何需要从原始素材（抓取文章、AI生成草稿、笔记）转化为寓言城堡发布级内容
- 用户贴了一段文字，要求"按我的风格重写"

## How It Works

### 核心冶炼算法

```
原始素材 → [反常识钩子] → [心理学×国学双视角] → [场景代入] → [金句收尾]
                │                  │                    │              │
          推翻一个常见认知    至少1个心理学概念      你走进办公室…   让人想截图的
                            至少1句国学经典                       bold sentence
```

### 风格DNA（不可偏移）

**语气**: 温柔但有力量，不说教。像朋友聊天，不像老师在讲课。
**定位**: 国学×心理×隐学×商业 — 不是搬运工，是冶炼者。
**差异化**: 别人讲"术"，你同时讲"道"。

### 写作结构

1. **开头（反常识钩子）** — 第一句就推翻读者认知
   - "你有没有发现…"
   - "你可能听过…但真相是…"
   - "所有人都在说X，但没人告诉你Y"

2. **中段（双视角冶炼）** — 心理学 + 国学，不是堆砌是融合
   - 心理学概念必须准确（认知失调、沉没成本、达克效应、损失厌恶…）
   - 国学引用必须贴切（道德经、庄子、王阳明、韩非子、鬼谷子…）
   - 场景化描述，不要抽象说教："你打车去公司，司机跟你聊了一路…"

3. **结尾（金句+出路）** — 讲残酷真相但给方向
   - 至少一句加粗金句
   - 变现方向 + 口播标题

### 绝对禁止

**禁用词**: 值得注意的是、首先/其次/最后、总的来说、综上所述、在这个信息爆炸的时代、随着科技的发展、不可否认、赋能、抓手、闭环、底层逻辑（过度使用）

**禁用行为**: 
- 列表式罗列（要自然叙述）
- 空洞鸡汤（必须有具体场景和方法）
- AI腔（读起来像教科书就是失败）

### 必须包含

- 反常识/颠覆性开头
- 心理学概念或国学引用
- 场景化描述
- 极端对比（数字或场景的强烈反差）
- 至少一句加粗金句
- 结尾: 变现方向 + 口播标题

## 冶炼标准（自检清单）

改写完成后逐条检查：
- [ ] 🚫 拒绝伪融合：国学×心理的连接是否有底层人性共通点？
- [ ] 🎯 拒绝信息冗余：每段是否提供了可验证的认知增量？
- [ ] ⚡ 操作化交付：读者看完知道做什么？
- [ ] 💎 矿石→金子：是原创认知框架，不是信息汇编？
