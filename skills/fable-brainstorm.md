# 冶炼选题脑暴

## When to Use

- 有一个话题但不知道从什么角度写
- 想找国学×心理×商业的交叉点
- 需要"景一式"的独特视角而不是老生常谈
- 用户说"帮我想想这个怎么写""有什么角度"

## How It Works

### 选题冶炼矩阵

每个选题从以下4个维度交叉扫描：

| 维度 | 拷问 | 国学钩子 | 心理锚点 |
|------|------|---------|---------|
| **人性底层** | 这个话题触碰了人性的哪个不变部分？ | 老子/庄子怎么说 | 哪个认知偏差在起作用 |
| **权力博弈** | 谁在获利？谁在受损？规则谁定的？ | 韩非子/鬼谷子框架 | 系统正当化理论 |
| **变现路径** | 这个认知值多少钱？怎么变成产品？ | 古代商人智慧 | 损失厌恶/稀缺效应 |
| **反常识** | 大多数人的认知错在哪？真相是什么？ | 王阳明"知行合一"的颠覆 | 达克效应/确认偏差 |

### 输出格式

对每个选题输出：
```
### 选题：{标题}
**反常识钩子**：{一句话颠覆认知}
**国学锚点**：{道德经/庄子/王阳明/韩非子… 的具体引用}
**心理锚点**：{具体心理学概念 + 如何应用}
**场景切口**：{从哪里写起让读者有代入感}
**金句雏形**：{一句有冲击力的结论}
**变现方向**：{这个内容能怎么变现}
```

## 冶炼禁忌

- 不要生拉硬拽：国学×心理的连接必须有底层人性共通点
- 不要重复别人的角度：同样的选题，景一的角度必须有信息增量
- 不要脱离场景：每个角度都要落到读者"今天能做什么"
